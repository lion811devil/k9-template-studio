# K9 Template Studio V5.8

Aggiunta sezione Fonti gratuite professionali.

La PWA ora apre direttamente fonti gratuite/compatibili da cui scaricare elementi HD/SVG.
Gli elementi scaricati vanno caricati come PNG personale oppure inseriti in assets/elements e registrati in elements.json.

Nota licenze: prima di usare ogni asset, controllare la licenza del singolo file.
