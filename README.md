# K9 Template Studio V8

Editor statico PWA per creare attestati/certificati professionali.

## Funzioni
- Modelli generati via codice: nessun template mancante.
- Caricamento foto soggetto.
- Testi modificabili.
- Elementi grafici: QR placeholder, coccarda, sigillo, linee, cornici.
- Esportazione PNG e PDF.
- Salvataggio locale nel browser.
- Import/export progetto JSON.
- Compatibile con GitHub Pages.

## Pubblicazione GitHub Pages
Settings → Pages → Source: Deploy from a branch → Branch: main → /(root) → Save.

## File principali
- `index.html`
- `css/style.css`
