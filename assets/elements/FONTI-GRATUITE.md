# Fonti gratuite consigliate

Usa solo file con licenza compatibile.

## Fonti più sicure
- FreeSVG: cerca asset CC0/pubblico dominio.
- Openclipart: generalmente pubblico dominio/CC0.
- SVG Repo: controlla la licenza del singolo file.
- Wikimedia Commons: usa preferibilmente Public Domain/CC0; se è CC-BY, conserva attribuzione.
