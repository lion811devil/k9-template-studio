# Fonti gratuite per elementi HD

Usa queste fonti per scaricare elementi SVG/PNG da integrare nella PWA.

## Consigliate

1. FreeSVG
- Licenza: CC0 / pubblico dominio.
- Cerca: certificate border, rosette, medal, badge, ribbon, dog, paw.

2. SVG Repo
- Molti file sono open-licensed e utilizzabili commercialmente.
- Controlla sempre la licenza del singolo file prima dell'uso.

3. Openclipart
- Clipart originali rilasciate in pubblico dominio/CC0.
- Ottimo per icone, impronte, badge semplici e simboli.

4. Wikimedia Commons
- Controlla sempre licenza e obblighi di attribuzione del singolo file.
- Usa preferibilmente file Public Domain o CC0.

5. Vecteezy / Freepik
- Usabili solo rispettando licenza, attribuzione e limiti del piano gratuito.
- Non caricare nella PWA asset con licenze che vietano redistribuzione o uso commerciale.

## Come integrarli

- Scarica SVG/PNG in alta qualità.
- Se è SVG, converti in PNG trasparente se necessario.
- Carica il file in `assets/elements/...`.
- Aggiungi una voce in `assets/elements/elements.json`.
